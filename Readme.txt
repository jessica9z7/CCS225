The Monster and Zombie Kingdom Adventure Game has five chapters inside the Game Modules folder which you might need Pycharm to run the program
There are 6 files called: main_game.py, Milestone3Chapter_1.py, Milestone3Chapter_2.py, Milestone3Chapter_3.py, Milestone3Chapter_4.py, Milestone3Chapter_5.py
The main_game.py file is the entry point of the game, which imports and runs each chapter module in sequence. Each chapter module contains the storyline, characters, and challenges specific to that chapter of the adventure game. Players progress through the chapters by completing tasks and making choices that affect the outcome of the game.

At first you open the project then go to the main_game.py file then double click and click on Run Python then click on Run Python File in Terminal
